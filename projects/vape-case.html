Vape Lock Case

Personal Project · Mechanical + Hardware · Work in Progress

This project explores a combined mechanical and hardware-based system designed to reduce vaping through physical constraint and behavioral feedback rather than pure restriction. The core idea is to place the vape inside a controlled enclosure that interfaces with a mobile device to regulate usage over time.

The system is designed to mechanically limit access to the device while also using hardware control to dynamically reduce intensity or fully disable the vape based on usage patterns, similar in spirit to phone screen-time controls.

Concept reference image representing enclosure form factor and interaction model.

System Concept

  • Mechanical enclosure to physically contain the vape device
  • Embedded hardware to interface with the vape’s power or control circuitry
  • Phone-connected control logic to regulate usage over time
  • Ability to reduce output intensity or fully disable the device

Current Focus

  • Mechanical design and tolerancing for the enclosure
  • Exploring hardware interfaces that do not permanently modify the vape
  • Defining safe, reversible control mechanisms
  • Designing a user interaction model focused on gradual reduction

This project is still in development, with current efforts focused on validating the mechanical and hardware architecture before moving toward tighter software integration.
